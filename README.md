# BriefAI — AI-Powered Pre-Meeting Sales Intelligence

> "Walk into every meeting knowing everything."

BITSoM × Masai School | Product Management with Generative & Agentic AI | Batch 2 | Problem Statement 4

**Live Demo:** https://briefai-eight.vercel.app

---

## What It Does

Enter a company name → BriefAI researches it using Claude AI → Get a 1-page meeting brief in seconds.

Built with React 18 + Vite + Anthropic Claude + Vercel

---

## API Keys Needed

**Anthropic API Key** (Required)
- Go to console.anthropic.com
- Sign up → Click API Keys → Create Key
- Copy the key (starts with sk-ant-...)

**NewsAPI Key** (Optional)
- Go to newsapi.org/register
- Sign up free → API key shown on dashboard
- Copy the key
